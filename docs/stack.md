# Stack Tecnológica do MercadoOS

## Frontend

- `Next.js` — framework React para aplicações modernas com SSR/SSG e roteamento baseado em arquivos
- `React` — biblioteca principal de UI
- `TypeScript` — tipagem estática para maior segurança e produtividade
- `Tailwind CSS` — utilitários de CSS para design consistente e escalável
- `shadcn/ui` — componentes de interface reutilizáveis e acessíveis

## Backend

- `NestJS` — framework Node.js modular para APIs escaláveis
- `TypeScript` — tipagem no backend e integração com NestJS
- `Prisma` — ORM moderno para modelagem e migrações de banco de dados
- `PostgreSQL` — banco de dados relacional robusto para regras de negócio e transações
- `Redis` — cache e filas em memória para desempenho
- `BullMQ` — processamento de filas para jobs assíncronos

## Infraestrutura

- `Docker` — containerização para ambientes consistentes
- `Docker Compose` — orquestração de ambiente local
- `pnpm` — gerenciador de pacotes rápido para monorepo
- `Turborepo` — build system para monorepo com cache inteligente
- `GitHub Actions` — CI/CD para build, teste e entrega contínua

## Qualidade e Ferramentas

- `ESLint` — análise estática de código e padronização
- `Prettier` — formatação de código automática
- `Jest` — testes unitários e integração
- `Playwright` — testes end-to-end
- `Swagger` — documentação de APIs (quando aplicável)
