# MercadoOS

MercadoOS é um ERP SaaS moderno para supermercados e varejo de proximidade.

## Visão do Projeto

Fornecer uma plataforma de gestão completa e escalável para operações de mercado, com foco em estoque, vendas, clientes, financeiro e relatórios operacionais.

## Estrutura do Monorepo

- `apps/` — aplicações principais (frontend e backend)
- `packages/` — pacotes compartilhados, libs de domínio e utilitários
- `docs/` — documentação de arquitetura, stack e roadmap
- `docker/` — configuração de containerização e ambiente local
- `specs/` — especificações de sprint e critérios de entrega
